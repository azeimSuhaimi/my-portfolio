Tentang Saya

Saya percaya bahawa setiap projek adalah peluang untuk belajar dan berkembang, oleh itu saya sentiasa berusaha untuk meningkatkan kemahiran teknikal saya dan sentiasa terbuka untuk mencuba teknologi baru. Matlamat saya adalah untuk menghasilkan aplikasi yang bukan sahaja memenuhi keperluan klien, tetapi juga memberikan pengalaman pengguna yang luar biasa.

Dengan latar belakang dalam reka bentuk web dan pembangunan frontend, saya mahir dalam teknologi seperti HTML5, CSS, JavaScript, serta framework seperti Bootstrap. Saya juga mempunyai pengetahuan dalam backend development menggunakan PHP dan Laravel, serta pengurusan pangkalan data menggunakan MySQL.
